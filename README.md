## 📋 Descripción

**Mapa interactivo comparativo** entre el **Modelo Psicosocial** y el **Modelo de Gestión de Casos** para la intervención en trabajo social. Desarrollado como parte del **Taller 1 COIL - Innovación En La Interacción Con Individuos**.

### ✨ Características Principales
- 🎨 **Interfaz moderna y responsive**
- 🔄 **Navegación interactiva** entre modelos
- 📱 **Optimizado para móviles** y desktop
- ♿ **Accesible** y fácil de usar
- 🎭 **Animaciones suaves** y profesionales

## 👥 Caso de Estudio: Juliana

<div align="center">

### 🌟 Perfil del Caso

</div>

El proyecto presenta el caso de **Juliana**, una adolescente de 15 años con embarazo de 2 meses que enfrenta múltiples desafíos:

| 🎂 **Edad** | 👶 **Situación** | 🏠 **Contexto Familiar** | 🎓 **Educación** |
|-------------|-------------------|---------------------------|-------------------|
| 15 años | Embarazo de 2 meses | Conflicto familiar | Riesgo de deserción escolar |

### 🚨 Desafíos Identificados
- 💔 **Conflicto familiar** - Reacción negativa de los padres
- 🎒 **Riesgo educativo** - Posible abandono escolar
- 👥 **Rechazo social** - Estigmatización por compañeros
- 🔮 **Incertidumbre** - Dudas sobre el futuro


### 3️⃣ Enfoque Integrado
```
🔄 Combinación de ambos modelos
📋 Estrategia comprehensiva
📈 Maximización de resultados
⚖️ Balance entre recursos y bienestar
```

## 🚀 Instalación y Uso

<div align="center">

### 📦 Guía de Instalación Rápida

</div>

#### 1️⃣ **Clona el repositorio**
```bash
# HTTPS
git clone https://github.com/tony-ssr/Taller-1-COIL.git

# SSH
git clone git@github.com:tony-ssr/Taller-1-COIL.git
```

#### 2️⃣ **Navega al directorio**
```bash
cd "Taller 1 COIL"
```

#### 3️⃣ **Inicia el servidor local**
```bash
# Con Python 3
python -m http.server 8000

# Con Python 2
python -m SimpleHTTPServer 8000

# Con Node.js (si tienes http-server instalado)
npx http-server -p 8000
```

#### 4️⃣ **¡Disfruta! 🎉**
Visita **[http://localhost:8000](http://localhost:8000)** en tu navegador
